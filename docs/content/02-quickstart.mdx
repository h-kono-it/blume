# Quickstart

<Steps>
  <Step title="Add content">
    Create `docs/index.mdx` with some frontmatter and Markdown.
  </Step>
  <Step title="Start the dev server">
    Run `blume dev`. Blume generates the runtime and serves your docs with hot
    reload.
  </Step>
  <Step title="Build for production">
    Run `blume build`. Static HTML is written to `dist/`, with a local search
    index built over it.
  </Step>
</Steps>

## A first page

```mdx
---
title: Introduction
description: Welcome to my docs.
---

# Introduction

Welcome! Use **Markdown** and built-in components like:

:::note
Blume ships callouts, cards, tabs, and more.
:::
```
