no imports required.

## Accordion

<Accordion title="Is the core theme React-free?">
  Yes. Interactive bits like tabs and search are vanilla custom elements. React
  is only enabled when you add an island.
</Accordion>

## API reference

`Endpoint`, `ParameterTable`, `RequestExample`, `ResponseExample`, and
`AuthMethod` power API docs. Generate them from an OpenAPI spec with
`blume import openapi`.
