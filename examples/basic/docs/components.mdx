# Components

Blume ships a set of MDX components you can use in any page.

<Callout type="info" title="Heads up">
  Callouts support info, tip, success, warning, and danger types.
</Callout>

<Callout type="warning">This API is experimental.</Callout>

<Steps>
  <Step title="Install">Add Blume to your project.</Step>

  <Step title="Ship">Run `blume build` and deploy.</Step>
</Steps>

## Tabs

<Tabs>
  <Tab title="bun">Run `bun add blume`.</Tab>
  <Tab title="npm">Run `npm install blume`.</Tab>

