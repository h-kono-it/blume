# Components

Blume ships a set of MDX components you can use in any page.

## Callouts

Callouts are written as directives — `:::note` (or `:::tip`, `:::success`,
`:::warning`, `:::danger`), with an optional `[title]`:

:::note[Heads up]
Callouts support note, info, tip, success, warning, and danger types.
:::

:::warning
This API is experimental.
:::

```package-install
blume
```

## Typography

Superscript and subscript: H~2~O is water; E = mc^2^ is mass–energy.

## Math

With `markdown.math` enabled, inline math like $a^2 + b^2 = c^2$ and block math
render with KaTeX:

$$
\int_0^1 x^2 \,dx = \frac{1}{3}
$$
